## Descripcion general de la practica:
En esta practica se hicieron los ejercicios de la guia que proponian realizar trabajos para aprender
sobre diseno de circuitos digitales a nivel de transistores, creando compuertas logicas para realizar
un par de funciones propuestas con ellas. Junto a esto se requerian conocimientos no solo de diseno,
sino tambien de algebra booleana y simplificacion de funciones logicas.
